Rol de Ansible: Actualizaciones Desatendidas
============================================

Rol de Ansible para configurar las actualizaciones desatendidas.

Libro de Ejemplo
----------------

```yaml
- hosts: servers
  roles:
      - cloudcxn.actualizaciones-desatendidas
```
